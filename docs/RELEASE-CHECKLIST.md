# Release checklist

## Build and verification

- [x] `pnpm run lint`
- [x] `pnpm run typecheck`
- [x] `pnpm run test`
- [x] `pnpm run test:e2e`
- [x] `pnpm run prepare:store`
- [x] `pnpm run build`
- [x] `pnpm run zip`
- [x] `pnpm run verify:release`
- [x] Confirm the generated zip hash is recorded below.

## Package audit

- [x] Manifest is MV3 and declares only the documented permissions.
- [x] Manifest has side panel, popup, background service worker, and icons.
- [x] Package contains no source maps or unbundled executable code.
- [x] Privacy and store listing documents are up to date.
- [x] Loaded Chromium smoke, responsive QA, accessibility audit, and fidelity ledger are green.

## Artifact

The reproducible artifact is generated by `pnpm run zip` under `.output/`. The verification script prints the exact path, byte size, and SHA-256 hash for the artifact used in a release.
