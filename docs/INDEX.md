# Documentation index

Read documents by route, not by directory scan.

| Document | Read when |
| --- | --- |
| `docs/WORKFLOW.md` | Changing state, task scope, checks, or commit workflow |
| `docs/PRODUCT.md` | Implementing or changing product behavior and MVP scope |
| `docs/DESIGN.md` | Implementing or visually verifying UI |
| `docs/ARCHITECTURE.md` | Changing entrypoints, data flow, storage, permissions, or module boundaries |
| `docs/ROADMAP.md` | Selecting the next task or changing dependencies/order |

The active route is always declared in `.project/state.json#activeTask.requiredDocs`.
